* **Q: Will The Long Haul be backported to versions before 1.20.1?**
  * **A:** TLH is only available for 1.20.1 and eventually will update to 1.21.X.

***

* **Q: Is The Long Haul gonna be available on** CurseForg&#x65;**?**
  * **A:** The Long Haul isn't currently available on CurseForge as I don't see a need nor do I want to go through the process of porting it but its not never.

***

* **Q: What's the hardware requirements to run The Long Haul?**
  * **A:** If your PC can run vanilla Minecraft, and you are able to allocate a minimum of 6GB of ram to TLH, you can run it. If you don't know how to allocate more RAM I will be adding a page soon showing how to do so.

***

**Q: Im on MacOS and the modpack wont open?**

* **A:** There is very simple solution, find [SplashScreen](https://modrinth.com/mod/splashscreen) and disable it as it is not compatible with MacOS  as it's code conflicts with apples operating software.

***

* **Q: I added a mod & I clicked update all & I enabled the disabled mods and now the modpack is not opening.**
  * **A:** Any and all disabled or older version mods are that way intentionally so that the modpack can work properly and seamlessly. If you want a mod to be part of the official pack feel free to open a ticket on TLH [github](https://github.com/rEiLLyy/The-Long-Haul-Vanilla-Refined/issues)! As well, you can join the [discord](https://discord.gg/kxJDajqf) to make suggestions or ask for help regarding anything with TLH.

***

* **Q: Does The Long Haul work with OptiFine?**
  * **A:** OptiFine will never be supported. TLH is already optimized using OptiFine adjacent methods.
